Covid tracker19

A small project of the data that the covid-19 leaves

Progress

100%

In this project I have tried to create a tracker of the data that the covid-19 virus has left in the countries around the world

For this project I use react as front-end and some packages as UI material and Charts.js.

To bring the data, use an API called covi19.mathdro which allows us to have data from various countries regarding covid-19
